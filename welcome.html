How NoPlugin works

NoPlugin searches web pages for embedded plugin objects. If the file type is supported by your web browser (MP4, WAV, MP3, etc.), NoPlugin will play it directly in the web page. If the file cannot be played by your web browser, NoPlugin can still download the file so you can open it with your computer's media player.

If a page says you don't have a plugin installed, right-click the page and select 'Toggle NoPlugin Compatibility Mode'. This will trick some pages into thinking you have Flash, QuickTime, Windows Media Player, and other plugins installed.

VLC Media Player is recommended for opening these files, since it will play just about every video and audio file in existence. Adobe Flash Player Projector is required to run Flash content.
